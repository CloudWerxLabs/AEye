# 🤖 AEye - AI Vision Chrome Extension

A powerful Chrome extension that combines the capabilities of Grok AI with computer vision to analyze images and provide intelligent responses.

## 🌟 Project Overview

AEye is an innovative Chrome extension that brings powerful AI capabilities directly to your browser, transforming how you interact with artificial intelligence.

## ✨ Key Features

- 🖼️ Image Analysis: Upload and analyze images using Grok's advanced vision model
- 💬 Natural Language Chat: Interact with Grok AI through a user-friendly chat interface
- 🎨 Modern UI: Beautiful gradient-based design with smooth animations
- 🔄 Context Awareness: Maintains chat history and scroll position
- 📱 Responsive Design: Works seamlessly across different screen sizes
- 🔒 Secure API Key Management
- 🌐 Browser-Wide Accessibility
